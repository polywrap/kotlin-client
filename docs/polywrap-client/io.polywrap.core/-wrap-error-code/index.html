WrapErrorCode

enum WrapErrorCode : Enum<WrapErrorCode>

Wrap error codes provide additional context to WrapErrors.

Error code naming convention (approximate): type of handler type of functionality piece of functionality ==> handler_typeFn_pieceFn

Error code map: 0 -> Invalid 1-25 -> Client 26-50 -> URI resolution 51-75 -> Wrapper invocation & sub-invocation 76-255 -> Unallocated
